# Hotel Management System

A web-based Hotel Management System built using Spring Boot, Java, HTML, CSS, and MySQL.

## Features
- User Login & Registration
- Room Booking
- Customer Management
- Admin Panel
- Secure Authentication

## Technologies Used
- Java
- Spring Boot
- HTML
- CSS
- MySQL

## How to Run
1. Clone the repository
2. Open in IntelliJ / VS Code / Eclipse
3. Configure MySQL database
4. Run the Spring Boot application
